Windows下安装JDK、Tomcat、Maven与配置

一、JDK的下载与环境变量配置

1.1 下载jdk1.8

官网链接：https://www.oracle.com/java/technologies/downloads/#java8-windows

根据系统选择对应版本即可

1.2 安装

双击下载好的安装包，建议更改一下安装路径，其他下一步即可，此处不在赘述。

1.3 配置环境变量

三、Maven的下载以及配置

3.1 Maven下载

官网：https://maven.apache.org/download.cgi
点击下载压缩包，并解压到指定目录下

注意：Maven的各版本对JDK的要求是不同的
Maven3.1+要求至少JDK1.7及以上版本
历史版本查看链接：https://maven.apache.org/docs/history.html

3.2 配置

3.2.1 配置环境变量和字符编码

字符编码：默认是当前系统中文GBK编码，需要将其设置成UTF-8

3.2.2 配置repo、jdk

Maven默认的repo在C:\Users\Administrator.m2文件夹里，需要修改默认设置
