Windows下IntelliJ IDEA的安装、配置与使用

一、IDEA的相关信息

IDEA，全称 IntelliJ IDEA，是 Java 语言的集成开发环境，IDEA 在业界被公认为是最好的 java 开发工具之一，尤其在智能代码助手、代码自动提示、重构、J2EE支持、Ant、JUnit、CVS 整合、代码审查、创新的 GUI 设计等方面的功能可以说是超常的。
官网下载地址：https://www.jetbrains.com/idea/download/#section=windows
官网提供的详细使用文档：https://www.jetbrains.com/help/idea/meet-intellij-idea.html

个人建议配置：内存 8G 或以上，CPU 最好 i5 以上，最好安装块固态硬盘(SSD)，将 IDEA
安装在固态硬盘上，这样流畅度会加快很多。

3、安装总结

从安装上来看，IntelliJ IDEA对硬件要求似乎不是很高。但是实际开发中其实不然，因为IDEA执行时会有大量的缓存、索引文件，所以如果你正在使用Eclipse/MyEclipse，想通过IDEA来解决计算机的卡顿、缓慢等问题，这基本上时不可能的，本质上应该对自己的硬件设备进行升级。

5、查看设置目录结构

这是IDEA的各种配置的保存目录。此设置目录有一个特性，就是当删除掉整个目录之后，重新启动IDEA会再重新自动生成一个全新的默认配置，所以如果把IDEA配置改坏了，可通过删除该目录，一切都会还原到默认。

5.1 config目录

config目录是IDEA个性化配置目录，或者说是整个IDE设置目录，此目录可看成最重要的目录，没有之一，在安装新版本IDEA会自动扫描硬盘上的旧配置目录，指的就是该目录。该目录主要记录：IDE主要配置功能、自定义的代码模板、自定义的文件模板、自定义的快捷键、Project的task记录等等个性化的设置。

5.2 system目录

system目录是IDEA系统文件目录，是IDEA与开发项目一个桥梁目录，里面主要有：缓存、索引、容器文件输出等，虽然不是重要的目录，但也是最不可或缺的目录之一。
